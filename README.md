# Strata Forge Website

This project is the marketing website for Strata Forge, built with **SvelteKit** and **Vite**.

## Development

1. Install dependencies:

    ```bash
    npm install
    ```

2. Start the development server:

    ```bash
    npm run dev
    ```

## Building

To create a production version of your app:

```bash
npm run build
```

## Tech Stack

- **Framework**: SvelteKit
- **Styling**: Custom CSS (`app.css`)
- **Icons**: Lucide Svelte
